Compact Observation Format

The Masterloop Compact Observation Format is a comma-separated UTF-8 text document. The general structure of this format is:

<observation_id>,<timestamp>,<value><cr><lf>

where:

observation_id Observation id number according to device template.
timestamp Timestamp of value formatted as an ISO 8601 string. If timestamp of observation is unknown, use “0”, which will take the time of the server at the time of logging.
value Value as string, according to observation data type defined for the device template:
  • Boolean: “0”=false, “1”=true
  • Double: NNNN.nnnn (note: point decimal separation symbol)
  • Integer: NNNNNN
  • Position (2D): LAT.DLAT,LON.DLON
  • Position (3D): LAT.DLAT,LON.DLON,ALT.DALT
  • String: Text up to 1023 characters.
